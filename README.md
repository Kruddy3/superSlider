# superSlider API
Iterate through lists with jQuery.

# Installation
 `$bower install superSlider`
# HTML

```HTML
<div id="slider">
  <button class="super-left"></button>
  <ul>
    <li></li>
    <li></li>
    <li></li>
  </ul>
  <button class="super-right"></button>
</div>
```

# JS

```javascript
$("#slider").superSlider({
  // how many elements you want shown
  display:4, //defaults to 5
});
```
